Использование механизма двойной записи - бухгалтерских проводок (общая информация)

Для отражения хозяйственных операций в программе используются документы, представляющие собой удобный инструмент для ввода данных. В момент проведения  информация из заполненной формы документа записывается в учетные данные в виде проводок, которые, в свою очередь, служат для построения отчетов. Таким образом, введенные в программу данные после обработки становятся доступны для пользователя в виде отчетов.

Принцип двойной записи

Принцип двойной записи предусматривает, что сумма хозяйственной операции одновременно заносится по двум счетам (в дебет одного и кредит второго). Данный подход обеспечивает достоверность учета и служит дополнительной защитой от расхищения собственности предприятия. Например, при получении 10 000 рублей по чеку в банке указанная сумма отражается по дебету счета "КАССА" и кредиту счета "БАНК". Если часть денег по дороге в кассу исчезла, недостача будет выявлена при первой же сверке.

Содержание проводок

Помимо суммы операции проводка в обязательном порядке содержит сведения о корреспондирующих счетах и субконто.
С помощью счетов обеспечивается группировка информации о хозяйственной деятельности предприятия.  Счета выбираются из плана счетов. Подробную информацию см. План счетов (общая информация).
Для записи аналитической информации в системе программ "1С:Предприятие" используются субконто. В программе к каждому счету «прикреплены» определенные виды субконто ("Имущество", "Номенклатура", "Контрагенты", "Сотрудники" и т.д.). Подробную информацию см. Виды субконто.

Создание проводок

Если в форме документа нажать кнопки "Провести" или "ОК", то программа, руководствуясь заложенной в нее методологией учета, автоматически создаст одну или несколько проводок. На практике обычно используется именно этот способ, поскольку предусмотренный в 1С:Управляющем набор видов документов учитывает специфику работы большинства предприятий.
Для тех редких ситуаций, когда подходящий для отражения хозяйственной операции вид документа отсутствует, предусмотрен ручной режим записи проводок. Чтобы им воспользоваться, необходимо  в меню "Хоз.операции" открыть "Общий журнал" и нажать кнопку "Добавить". В выпадающем списке нужно выбрать строку "Хоз.операция (ручной ввод)". Подробную информацию см. Хоз.операция (ручной ввод).

Использование проводок для формирования отчетов

На основании проводок в программе можно формировать отчеты. В 1С:Управляющем для каждого вида анализируемых данных (деньги, имущество, расчеты с контрагентами, сотрудниками и т.п.) используются отдельные отчеты. Это позволяет отражать информацию в более удобном для восприятия виде.
Вместе с тем пользователям, имеющим опыт работы в бухгалтерских программах, предоставлена возможность просматривать сведения в привычном для них формате (например, в виде оборотно-сальдовой ведомости или карточки счета). Для получения стандартных бухгалтерских отчетов служит меню "Сервис – Внешние обработки". Подробную информацию см. Внешние обработки.
