Работа с формой счета

Форма счета предназначена для изменения реализованного в 1С:Управляющем плана счетов путем добавления новых счетов и корректировки существующих (предопределенных). При обращении к форме счета пользователю желательно ограничиться просмотром информации, поскольку любые активные действия могут в дальнейшем обернуться проблемами при работе с программой. В частности, все документы, проводки и отчеты настроены на использование исключительно предопределенного плана счетов. По этой причине вносить в базу первичные данные при помощи измененных счетов допускается только посредством документа «Хоз.операция (ручной ввод)», а для просмотра операций — применять исключительно внешние (бухгалтерские) отчеты.

Кроме того, для корректировки предопределенных счетов (субсчетов) необходимо снять конфигурацию с поддержки. В этом случае пользователь лишается возможности получать обновления программы и принимает на себя ответственность за некорректное функционирование 1С:Управляющего. Общую информацию см. План счетов (общая информация).

Управление иерархией счетов

Реализованный в 1С:Управляющем план счетов поддерживает многоуровневую иерархию «счет – субсчета», то есть к некоторым счетам (субсчетам) открываются субсчета более низкого уровня. На счете консолидируются сальдо и обороты (дебетовые и кредитовые) по всем «подчиненным» ему субсчетам.
Для управления иерархией в форме счета предусмотрено поле «Подчинен счету». Для счетов самого высокого уровня (групп счетов) данное поле остается незаполненным. 

Представление счета

Реквизиты «Код», «Наименование» и «Представление в управленческом балансе» в 1С:Управляющем подобраны таким образом, чтобы, с одной стороны, соответствовать привычным для специалистов понятиям, а с другой - облегчить восприятие данных людьми, не являющимися профессиональными бухгалтерами.
В частности, для краткого написания счетов используются не цифровые комбинации, а интуитивно понятные буквенные обозначения (например, счету «Прибыль от основной деятельности» присвоен код «ПРИБЫЛЬ.ПЛЧ.ОСН»).

Реквизит «Представление в управленческом балансе» для большинства счетов совпадает с «Наименованием». Исключение сделано для счетов, входящих в группу «Закрывающиеся счета». В программе они используются для выполнения регламентных операций, на что прямо указывает представление в балансе и самой группы («Разница между активом и пассивом»), и отдельных счетов (скажем, сальдо по счету «Расходы общие» отражается как статья баланса «Не распределены общие расходы»). 

Свойства счета

Для указания свойств счета в 1С:Управляющем используются реквизиты «Вид», «Забалансовый», «Количественный» и «Валютный».

В поле «Вид» предусмотрено три возможных значения реквизита: активный, пассивный и  активно-пассивный.
На активных счетах накапливается информация о средствах предприятия (пример - счета «Имущество», «Товарно-материальные ценности», «Операции по кассам»). Поскольку средства не могут иметь отрицательную стоимость, появление отрицательного дебетового сальдо сигнализирует о допущенной ошибке при вводе данных.
Для пассивных счетов допускается наличие только положительного или нулевого кредитового сальдо. В финансовом учете данные счета в основном применяются для отражения источников формирования средств предприятия.
Последняя категория счетов, активно-пассивные, применяется для  отражения расчетов с контрагентами, а также для выполнения регламентных операций (главным образом, расчета прибыли).

Включив соответствующие флажки (галочки), можно установить следующие реквизиты счета:

  • Забалансовый — отражаемые по данному счету данные не включаются в  баланс (пример - товары, принятые на ответственное хранение);
  • Количественный — на счете регистрируется не только стоимость имущества, но и его количество. Данный подход в 1С:Управляющем не распространяется на счет «Имущество»: объекты учета на нем отражаются не в натуральном выражении, а по инвентарным номерам.
  • Валютный — данный реквизит устанавливается для счетов, предназначенных для учета денег и учета расчетов с контрагентами и сотрудниками.

Список видов субконто

Табличная часть формы предназначена для составления списка назначенных счету видов субконто (видов аналитики).  Подробную информацию см. Виды субконто.

Колонки списка:

  • Служебная - порядковый номер вида субконто. В 1С:Управляющем допускается к каждому счету «прикреплять» до 3-х видов субконто.
  • Вид субконто - наименование вида субконто. Видами субконто обычно являются справочники программы, реже - перечисления и виды документов.
  • Только обороты – флажком обозначаются виды субконто, которые нецелесообразно (или невозможно) задействовать для анализа сальдо по конкретному счету. Например, руководителю предприятия важно отслеживать поток платежей по статьям движения денег, однако в отношении остатков средств в кассе или на расчетном счете данный аналитический разрез применять бессмысленно.  

Изменение перечня видов субконто приводит к тем же результатам, что и корректировка прочих реквизитов счета, а именно - к сбоям в работе документов и отчетов. Однако опытные пользователи после снятия конфигурации с поддержки могут внести поправки в список видов субконто посредством имеющегося набора кнопок.

Кнопки управления списком:

 - добавление в список нового вида субконто

 - добавление новой строки путем копирования выделенной строки

 - замена выделенного вида субконто

 - удаление выделенной строки из списка
